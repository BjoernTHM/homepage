Chat-Client

The Chat-Client in Java... Once only an excersize starting to fascinating me.
This projekt where a co-work with my collegue Björn Pfarr. We started with learning Input-Output-Programming, Socket-Programming and trying to find an answer on: "How can I realize unsychronous network communication?"...
After searching and trying a while, we used 2 single Threads: 1 for incoming and 1 for outgoing messages/connections.

FTP-Client

"I want my own FTP-Client..." and so started the project "FTP-Client".
The program is using 2 Sockets. One for the Control-Stream and 1 for the Data-Stream. Everything else are string-commands using the Control-Stream.

Music-Organisation

"Organizing the music.. Without doing anything?." This program makes it possible.
All your music will be automatic sorted by Interpret -> Album -> Musicfile (as long as there are mp3-Tags inside the files).
